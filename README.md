# Training Evaluation Module

An activity module designed to streamline training evaluations and assessments. This plugin enables instructors to create structured training programs with customizable
sections and items, allowing for detailed evaluation and tracking of training evaluation.

## Use Cases

- Structured competency assessments
- Practical evaluations
- Work-integrated learning assessments

## Basic Usage

### Creating a Training Evaluation Activity

1. Turn editing on in your course
2. Click "Add an activity or resource"
3. Select "Training Evaluation"
4. Configure the activity settings and save

### Managing Training Sections

1. Open the Training Evaluation activity
2. Click "Manage Sections" to add, edit, or delete sections
3. Add items to each section

### Evaluating Students

1. Access the Training Evaluation activity
2. Click "Evaluate Students"
3. Select a student to evaluate
4. Complete the evaluation items for each section
